# homepage

Landing page for [apps.andymolenda.com](https://apps.andymolenda.com). Lists all projects with live links.

## Updating

When adding a new project, add a `<li>` entry following the existing pattern:

```html
<li><a href="https://<name>.apps.andymolenda.com"><name></a> &mdash; <span style="color:#9a9686;font-size:0.9rem;">short description</span></li>
```
