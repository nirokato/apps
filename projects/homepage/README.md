# homepage

Landing page for [apps.andymolenda.com](https://apps.andymolenda.com). Lists all projects with live links.

## Updating

When adding a new project, add a card entry following the existing pattern:

```html
<li><a class="project" href="https://<name>.apps.andymolenda.com">
  <div class="project-name"><name></div>
  <div class="project-desc">Short description</div>
</a></li>
```
