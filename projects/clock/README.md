# clock

World clock displaying current time across multiple timezones.

**Live:** [clock.apps.andymolenda.com](https://clock.apps.andymolenda.com)

## Timezones

- US/Mountain (America/Denver)
- UTC
- Asia/Tokyo

## Tech

Single-file static HTML/CSS/JS. Uses `Intl.DateTimeFormat` for timezone handling, JetBrains Mono for time display. Updates every second.
